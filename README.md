
# Personal Google Drive Clone 📁

A web-based file management application inspired by Google Drive, built with Node.js, Express, and MongoDB.

## Features ✨

- **User Authentication** - Secure registration and login with JWT tokens
- **Password Encryption** - Bcrypt-based password hashing for enhanced security
- **EJS Templating** - Dynamic server-side rendering for responsive UI
- **Data Validation** - Input validation using express-validator
- **Session Management** - Cookie-based session handling

## Tech Stack 🛠️

- **Backend**: Express.js (Node.js)
- **Frontend**: EJS Templates 
- **Database**: MongoDB with Mongoose ODM
- **Authentication**: JWT (JSON Web Tokens)
- **Security**: Bcrypt for password hashing
- **Development**: Nodemon for hot-reload
